Protein Structure Prediction of Human p53 Protein

Overview

This project focuses on predicting the three-dimensional (3D) structure of the Human p53 protein using computational modeling techniques. The predicted protein structure was analyzed and visualized using molecular visualization software to understand its structural features.

Objectives

- Predict the 3D structure of the Human p53 protein.
- Analyze the predicted protein model.
- Visualize the protein structure using PyMOL.

Tools Used

- SWISS-MODEL – Protein structure prediction
- PyMOL – Molecular visualization
- FASTA Sequence – Protein sequence input

Project Files

- "p53.fasta" – Human p53 protein sequence
- "model_01.pdb" – Predicted 3D protein structure
- "visualization of model_01.pdb" – PyMOL visualization
- "SWISS-MODEL results" – Model prediction results
- "Final rendered structure" – High-quality rendered protein image

Workflow

1. Retrieved the Human p53 protein FASTA sequence.
2. Submitted the sequence to SWISS-MODEL for structure prediction.
3. Downloaded the predicted PDB structure.
4. Visualized the structure using PyMOL.
5. Rendered the final protein structure for analysis and presentation.

Results

The predicted 3D structure of the Human p53 protein was successfully generated and visualized. The resulting model provides a structural representation that can be used for further computational studies such as molecular docking, molecular dynamics simulations, and interaction analysis.

Future Scope

- Model validation using structural assessment tools.
- Molecular docking with potential ligands.
- Molecular dynamics simulations to evaluate structural stability.
- Protein–ligand interaction analysis.
